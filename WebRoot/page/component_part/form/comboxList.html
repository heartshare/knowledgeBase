下拉多选框--效果展示 (通过另外一种方式创建)

使用解释

引用的JS和CSS

下拉数据源(JSON格式) 

			var dimExternalSysType = [
				{"val":20001,"pid":-1,"txt":"BOSS","leaf":true},
				{"val":20002,"pid":-1,"txt":"CRM","leaf":true}
			];

使用(无需额外的调用JS) 

			<div id="externalSys" name="externalSys" class="AI-Comp"
				type="combox" ds="dimExternalSysType" multisel="true" showall="true"
				alltxt="--请选择-" allval="-1"></div>

属性说明 

			class		必须为AI-Comp
			ds 			JSON数据ID
			type		组件的类型
						combox	表示下拉框
			allowFilter	是否允许过滤
			showall
			alltxt		未选择时的中文展示
			allval		未选择时的value值
			multisel	是否多选
			showstyle	显示类型
						group 	分组显示
						tree	树形显示
			cascade		联动对应的下级ID
 			width           组件长度
                        value           默认值
            panelwidth      面板长度(为了保持页面美观,面板长度=组件长度+10)

方法说明 

                       创建对象        var comtmp = new AI.Comp.Combox({
			                               el:'combox1',					
			                               showAll:true,
			                               ds:arr,//eval("[{txt:'M902',val:58920140},{txt:'T105',val:86202159}]"),		                               						       	       multisel : true,
			                               allowfilter : true,
			                               value:arrayDimLevel2Str,
			                               id:'combox1'
		                          });
                       获取对象        AI.getComp('id').getValue()
 
 
                       获取对象的值    AI.getComp('id').getValue().val (单个) 或AI.getComp('id').getValue()[i].val (数组需要遍历)
                       销毁对象        AI.getComp('id').destory()
