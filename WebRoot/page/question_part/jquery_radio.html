用jquery通过name属性获取radio值有两种方式
方式A:
function getValue(){
		alert($("input[name=sex][checked]").val());
} 
方式B:
function getValue(){
		alert($("input[name=sex]:checked").val());
}  

经过测试人员测试(jquery版本:jquery-1.3.2.min.js):
针对方式A:在IE8、IE9的非兼容模式下,获取不了值 
针对方式B:在IE7、IE8(非兼容模式、兼容模式)、IE9(非兼容模式、兼容模式) ,均可以获取值

结论:
不要用方式A,使用方式B
